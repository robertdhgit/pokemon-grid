# Pokemon Grid
This is a pure frontend application built with Vite, React, Typescript and Styled Components, designed to display and filter customer data, showcasing best practices in performance, responsiveness, test coverage and code quality.

## 1. Local Environment Setup
This project uses NVM (Node Version Manager) to enforce a specific Node.js version, ensuring environment consistency across all developer machines.

### 1.1 Node.js Version
Before installing dependencies, please ensure you have NVM installed and running.

Run the following command in the project root directory to automatically switch to the required Node.js version:
```
nvm use
```
Required Version: 24.11.1

## 2. Running the Project

### 2.1 Install Dependencies
After confirming the correct Node version, install project dependencies:
```
yarn
```

### 2.2 Start the Development Server
Start the local development server:
```
yarn dev
```
Start the local development server: http://localhost:5173

### 2.3 Test the App:
After install the dependencies, run command:
```
yarn test
```
